SYNOPSIS

SYNRC FORM is a pure Erlang forms protocol for enterprise software. It suppors following target formats: XML, JSON, HTML, BML, SVG, JavaScript, BERT.

INTRO

FORM application provides set of CSS stylesheets for compact forms definitions and also it provides database model for storing metadata information about documents, fields and validations.

FORM

The forms on this HTML page are rendered by using N2O and NITRO from ns.synrc.com:8002 websocket endpoint.
