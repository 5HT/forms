FORM

INTRO

The forms module provides AST transform of FORM model into NITRO model (second stage) which later will be transformed into HTML5 encoding (first stage).

API

The FORMS module expects forms to expose the following API.

Module:id() -> tuple().

Should return the default instance of this document class.

Module:new(Name :: string(), Form :: #document{} | #element{}, Options :: list({atom(),term()})) -> element().

Generates NITRO object model from FORM.

Module:doc() -> binary() | list().

Retrieves the documentation string about this form module.
